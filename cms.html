CMS

콘텐츠관리시스템 : Content Management System

웹 페이지에 대한 콘텐츠 및 게시판, 기타 정보 등을 효율적으로 관리할 수 있도록 사용자 및 관리자에게 다양한 인터페이스를 제공하며,
JAVA Framework을 사용하여 추후 타 시스템과의 연동이 원활하도록 표준 API 기능을 제공합니다.

  • 사이트 · 메뉴관리

    #메뉴 생성 #메뉴이동 #CSS 생성 관리 기능

  • 콘텐츠 관리

    #스킨적용 #권한설정 #실명인증

  • 게시판 관리

    #스킨 적용 #권한설정 #실명인증

  • 편집기능

    #WYSIWYG #CSS지원 #자체 인쇄

  • 회원관리

    #회원 정보 관리 #조직도 관리

  • 통계관리

    #방문통계 #컨텐츠별 접속자 통계

사이트 · 메뉴관리

효율적인 사이트 · 메뉴 관리를 위해 다음과 같은 관리기능을 제공합니다.

  • 생성 · 수정 · 삭제 · 언어(K/E)
  • 메뉴 관리기능
  • 메뉴코드와 컨텐츠 분리
  • 메뉴 트리구조 지원
  • 메뉴별 HTML, 게시판, 갤러리, 동영상, 링크, 프로그램 연계

콘텐츠 관리

직관적이고 감각적인 콘텐츠 관리를 위해 여러 기능을 제공하고 있습니다.

  • 배너 및 팝업존 관리
  • 배너 및 팝업존 게시기간 설정
  • 이미지 관리 기능
  • 관리자 지정 기능
  • 웹에디터 컨트롤에 의한 자료 구축
  • 인쇄 · 스크랩, 확대 · 축소 기능
  • CMS통한 자료구축 및 기본 컨텐츠 제공

게시판 관리

관리자와 사용자, 모두의 쾌적한 게시판 이용을 위한 기능을 제공합니다.

  • 게시판 관리 기능
  • 일반게시판 권한 및 컨텐츠 연계가능
  • 포토게시판 권한 및 컨텐츠 연계가능
  • 동영상게시판 권한 및 컨텐츠 연계가능
  • 자동생성 및 스킨 적용 기능
  • 게시판별 IP차단, 불량단어 차단 기능
  • 권한별 읽기 · 쓰기 · 답변 제한기능
  • 머리말 · 꼬리말 · css파일지정
  • 실명인증 연동 가능
  • 새글표시, IP출력표시, 첨부파일 갯수, 첨부파일용량 설정

편집기능

편리한 홈페이지 관리를 위한 다양한 편집기능을 제공합니다.

  • Open API 방식으로 자동 다운로드 후 자동 설치
  • WYSIWYG 방식 제공
  • 다국어 지원 가능
  • Speller 기능이 지원되어 맞춤법 기능
  • 셀나누기 · 합치기와 같은 표기능
  • 차트 만들기 기능
  • 이미지, 동영상 등 멀티미디어 파일 삽입 · 처리
  • 스타일시트(CSS) 지원
  • DOC, HWP, XLS, PPT, GUL 파일을 불러와 HTML로 저장
  • 인쇄 시 자체 인쇄 기능, 미리보기 지원

회원관리

효율적인 회원관리를 위한 기능을 제공합니다.

  • 권한별 회원 정보 관리
  • 각 부서 및 직원관리, 조직도 관리

통계관리

방문자 분석을 통해 사이트 운영 및 마케팅의 방향을 설정할 수 있습니다.

  • 회원통계, 방문통계, 게시물 통계
  • 각 컨텐츠별 접속자 통계
  • 통계자료 엑셀 · PDF 저장
